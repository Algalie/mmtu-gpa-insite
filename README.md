# MMTU GPA Insite

A web-based GPA calculator and predictor for Mampong Technical University students.

## Features
- Student registration and authentication
- GPA calculation with reference modules
- Result saving and management
- Responsive design for all devices

## Technology Stack
- Python Flask
- SQLite Database
- HTML/CSS/JavaScript
- Responsive Design

## Deployment
This app is deployed on Render.com
